# QuickMock
基于Express的快速mock平台，无需配置数据库，启动后即可实现本地mock接口数据。通过接口url,返回对应接口json数据。

## 启动

``` bash
# install dependencies
npm install

# 访问localhost:3000/list
npm start

```

## 功能点
* 支持保存多个项目的接口数据
* 根据接口名称或url模糊查询接口
* 支持重新编辑以保存接口
* 创建接口后以json文件保存在本地
* 编辑接口数据实时预览及错误提示
